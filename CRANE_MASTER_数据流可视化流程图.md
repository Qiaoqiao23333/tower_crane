```
  Bit 0: Switch On
  Bit 1: Enable Voltage
  Bit 2: Quick Stop
  Bit 3: Enable Operation

  Bit 1: Switched On
  Bit 2: Operation Enabled
  Bit 3: Fault
  Bit 5: Quick Stop
```

## 5. 数据格式转换流程

```
┌─────────────────────────────────────────────────────────────────┐
│                     用户输入：90.0°                              │
└─────────────────────────────┬───────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│              angle_to_position() 转换函数                        │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 输入参数：                                                  │ │
│  │   angle = 90.0 (度)                                        │ │
│  │   gear_ratio = 10.0 (减速比)                               │ │
│  │   target_units_per_rev = 10000 (每圈命令单位)              │ │
│  ├───────────────────────────────────────────────────────────┤ │
│  │ 计算过程：                                                  │ │
│  │   units_per_degree = target_units_per_rev / gear_ratio / 360│ │
│  │                    = 10000 / 10 / 360                      │ │
│  │                    = 2.7778                                │ │
│  │                                                            │ │
│  │   position = angle × units_per_degree                      │ │
│  │            = 90 × 2.7778                                   │ │
│  │            = 250                                           │ │
│  ├───────────────────────────────────────────────────────────┤ │
│  │ 输出结果：                                                  │ │
│  │   position = 250 (命令单位, int32_t)                       │ │
│  └───────────────────────────────────────────────────────────┘ │
└─────────────────────────────┬───────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│           二进制编码（Little Endian）                            │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 十进制：250                                                │ │
│  │ 十六进制：0x000000FA                                        │ │
│  │ 二进制：00000000 00000000 00000000 11111010                │ │
│  ├───────────────────────────────────────────────────────────┤ │
│  │ Little Endian 字节序（低字节在前）：                        │ │
│  │   Byte 0: 0xFA (250 & 0xFF)                               │ │
│  │   Byte 1: 0x00 ((250 >> 8) & 0xFF)                        │ │
│  │   Byte 2: 0x00 ((250 >> 16) & 0xFF)                       │ │
│  │   Byte 3: 0x00 ((250 >> 24) & 0xFF)                       │ │
│  └───────────────────────────────────────────────────────────┘ │
└─────────────────────────────┬───────────────────────────────────┘
                              │
```
